# Tabela_Brasileirao
Projeto Scrapy para coleta da tabela do Brasileirão no site da CBF

Versão do python utilizada no Desenvolvimento: Python 3.8.3

Para rodar o BOT utilize o comando:
```
scrapy crawl brasileirao
```
Requerimentos
- scrapy
